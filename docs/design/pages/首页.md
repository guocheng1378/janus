# 首页 (HomePage) 设计规范

## 布局结构

- 顶部：可折叠大标题栏，标题 "首页"（`TopAppBar`（顶部大标题栏） + `MiuixScrollBehavior`（标题栏跟随滚动自动折叠））
- 内容：垂直滚动列表（`LazyColumn`（按需加载的滚动列表）），水平内边距 12dp（dp 是与屏幕密度无关的尺寸单位）
- 列表中唯一内容：`StatusCard` 组件，上下各 12dp 外边距

## 组件

| 组件 | 用途 |
|------|------|
| `TopAppBar` | 可折叠大标题栏，同时设置 `title` 和 `largeTitle` |
| `StatusCard` | 状态卡片组，展示模块激活状态和 Root 状态 |

### StatusCard 组件（定义于 StatusCard.kt）

`StatusCard` 是一个纵向排列的 `Column`（纵向布局容器），包含两个 `StatusItem`，间距 12dp。

**StatusItem 结构：**

- 外层：`Row`（横向布局容器），宽度填满，圆角 16dp，纯色背景，内边距 20dp
- 左侧：图标（`Icon`（图标组件）），28dp，白色
  - 正常状态：`MiuixIcons.Ok`
  - 异常/加载中：`MiuixIcons.Report`
- 图标与文字间距：16dp
- 右侧：`Column`
  - 标题：`headline2`（二级标题样式） 样式，白色，加粗
  - 摘要（可选）：`body2`（正文样式） 样式，白色 80% 透明度，标题下方间距 2dp

## 视觉规范

### 状态背景颜色

| 状态 | 背景色 | 色值 |
|------|--------|------|
| 正常/已激活 | 深绿色 | `#1B5E20` |
| 异常/未激活 | 深红色 | `#B71C1C` |
| 检测中（加载） | 深灰色 | `#424242` |

### 模块状态项（第一张卡片）

- **已激活**：绿色背景，Ok 图标，标题 "模块已激活"，有摘要
- **未激活**：红色背景，Report 图标，标题 "模块未激活"，有摘要

### Root 状态项（第二张卡片）

- **已获取**：绿色背景，Ok 图标，标题 "Root 权限正常"，有摘要
- **未获取**：红色背景，Report 图标，标题 "无 Root 权限"，有摘要
- **检测中**：灰色背景，Report 图标，标题 "正在检测 Root..."，无摘要

## 更新通告弹窗

首页承载版本更新通告弹窗（`SuperDialog`（弹窗组件）），与 Scaffold 同级放置。

- **触发条件**：Debug 构建每次启动弹出；Release 构建仅当 `versionCode` 大于上次记录值时弹出一次
- **标题**：`"v{版本号} 更新内容"`
- **摘要**：更新日志（多行文本）
- **QQ 群卡片**：`Card`（卡片容器） 包裹 `SuperArrow`（带箭头的可点击项），标题"加入 QQ 群反馈交流"，摘要显示群号，点击跳转加群链接
- **爱发电卡片**：`Card` 包裹 `SuperArrow`，标题"爱发电支持"，`endActions` 显示链接 `ifdian.net/a/janus`，点击打开爱发电页面
- **按钮**：`TextButton`（文字按钮） "知道了"，`fillMaxWidth`（撑满宽度），使用 `textButtonColorsPrimary()`（主题色按钮样式） 主色调
- **关闭行为**：点击"知道了"或点击弹窗外部均关闭，同时记录当前 `versionCode`

弹窗状态（`showUpdateDialog`）和关闭回调（`onDismissUpdateDialog`）由 `MainScreen` 传入。

## 交互

- 页面支持垂直滚动到边缘时的回弹效果（overScrollVertical）
- 标题栏支持滚动折叠
- 状态卡片仅展示，无用户操作
- 更新通告弹窗可通过点击"知道了"或点击外部关闭
