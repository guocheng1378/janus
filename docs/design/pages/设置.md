# 设置 (SettingsPage) 设计规范

## 布局结构

- 顶部：可折叠大标题栏，标题 "设置"（`TopAppBar`（顶部大标题栏） + `MiuixScrollBehavior`（标题栏跟随滚动自动折叠））
- 内容：垂直滚动列表（`LazyColumn`（按需加载的滚动列表）），水平内边距 12dp（dp 是与屏幕密度无关的尺寸单位）

## 组件

| 组件 | 用途 |
|------|------|
| `TopAppBar` | 可折叠大标题栏 |
| `Card` | 菜单容器，上边距 12dp，下边距 6dp |
| `SuperSwitch` | "隐藏桌面图标" 开关 |
| `SuperArrow` | "关于" 导航项，摘要 "Janus" |
| `SuperDialog` | 隐藏图标确认弹窗 |

## 视觉规范

- 第一个 Card（隐藏图标）：上边距 12dp，下边距 6dp
- 第二个 Card（关于）：下边距 6dp
- SuperSwitch 摘要根据状态切换（隐藏/显示）
- SuperArrow 点击后导航至 AboutPage
- 确认弹窗包含取消和确认按钮

## 关于页面 (AboutPage)

### 布局结构

- 顶部：`SmallTopAppBar`（固定小标题栏），标题 "关于"，带返回按钮（`MiuixIcons.Back`）
- 内容：垂直可滚动（`verticalScroll`）的 `Column`（纵向布局容器），水平居中

### 组件

| 组件 | 用途 |
|------|------|
| `SmallTopAppBar` | 固定顶部栏 + 返回按钮（`IconButton`（图标按钮） + `MiuixIcons.Back`） |
| `Image`（图片组件） | 应用图标，96x96dp，从 `PackageManager`（系统应用管理器） 动态获取 |
| `Text` | 应用名，`headline1`（一级标题样式） 样式，`onBackground`（前景色） 颜色 |
| `Text` | 版本号，`body2`（正文样式） 样式，`onSurfaceContainerVariant`（次要前景色） 颜色 |
| `Card` | 链接容器，水平内边距 12dp |
| `SuperArrow` x 4 | "项目"、"作者"、"赞助支持"、"QQ 群" 链接项 |

### 视觉规范

- 顶部间距 48dp
- 图标下方间距 16dp
- 版本号格式：`"v${BuildConfig.VERSION_NAME}"`，动态读取构建版本号（非硬编码）
- 链接卡片上方间距 32dp
- "项目"：`endActions`（右侧内容） 显示灰色文字 "GitHub"，点击打开仓库链接
- "作者"：`endActions` 显示灰色文字 "YunPeng"，点击打开作者 GitHub 主页
- "赞助支持"：`endActions` 显示灰色文字 "爱发电"，点击打开 `https://ifdian.net/a/janus`
- "QQ 群"：`endActions` 显示灰色文字 "119655862"，点击打开 QQ 群链接，长按复制群号到剪贴板并 Toast（屏幕底部短暂弹出的提示消息） 提示"群号已复制"（使用 `Box`（布局容器） + `combinedClickable`（同时支持点击和长按） 包裹 `SuperArrow`（带箭头的可点击项））

### 图标加载

- Preview（Compose 界面预览功能） 模式：创建空白 96x96 `ImageBitmap`（位图图片格式）
- 运行时：通过 `context.packageManager.getApplicationIcon()` 获取，根据屏幕密度转换为合适像素大小的 Bitmap

## 交互

- 设置页开启 "隐藏桌面图标" -> 弹出确认对话框 -> 确认后隐藏图标
- 设置页关闭 "隐藏桌面图标" -> 直接恢复图标显示
- 设置页点击 "关于" -> 导航到关于页面
- 关于页点击返回 -> 返回设置页
- 关于页点击 "项目" -> 打开 `https://github.com/penguinyzsh/janus`
- 关于页点击 "作者" -> 打开 `https://github.com/penguinyzsh`
- 关于页点击 "赞助支持" -> 打开 `https://ifdian.net/a/janus`
- 关于页点击 "QQ 群" -> 打开 `https://qm.qq.com/q/UJBp9bNnIQ`
- 关于页长按 "QQ 群" -> 复制群号 "119655862" 到剪贴板，Toast 提示
