# 功能 (FeaturesPage) 测试用例

## 页面概述

功能页面用于配置背屏高级功能（Tab 2（第三个标签页）），包含以下功能模块：统计上报拦截、背屏天气卡片（含刷新间隔和优先级子设置）、歌词显示（淡入时长和检测阈值）、壁纸相关（导航入口）、背屏常亮（含唤醒间隔设置）、背屏 DPI 设置、投屏旋转和投屏常亮。壁纸相关功能（壁纸防打断、壁纸锁定、AI 壁纸自定义）已拆分到壁纸子页面（WallpaperPage）。所有设置通过 `SharedPreferences`（应用本地配置文件）（`WhitelistManager`（白名单管理工具））持久化。

**源文件**: `app/src/main/kotlin/org/pysh/janus/ui/FeaturesPage.kt`
