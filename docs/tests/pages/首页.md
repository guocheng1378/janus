# 首页 (HomePage) 测试用例

## 页面概述

首页是应用的默认页面（Tab 0（第一个标签页）），启动后默认展示。页面通过 `StatusCard`（状态卡片组件）展示模块激活状态和 Root（超级管理员权限）权限状态，卡片颜色随状态变化。

**源文件**: `app/src/main/kotlin/org/pysh/janus/ui/HomePage.kt`
**关联文件**: `app/src/main/kotlin/org/pysh/janus/ui/StatusCard.kt`

## 测试用例

### 导航

#### TC-HOME-001: 默认页面

- **前置条件**: 应用已激活（已通过激活流程）
- **操作**: 启动应用
- **预期结果**:
  - 首页为默认显示页面（Tab 0）
  - 底部导航栏"首页"选项高亮

### 状态卡片展示

#### TC-HOME-002: 模块已激活 - 绿色卡片

- **前置条件**: Xposed（一种 Android 系统级的框架，可修改其他应用行为）模块已在 LSPosed（Xposed 框架管理器）中启用
- **操作**: 进入首页，观察模块状态卡片
- **预期结果**:
  - 卡片背景为绿色（#1B5E20）
  - 显示 Ok 图标
  - 标题显示"模块已激活"
  - 摘要显示"Hook（拦截并修改目标应用行为）工作正常"

#### TC-HOME-003: 模块未激活 - 红色卡片

- **前置条件**: Xposed 模块未在 LSPosed 中启用
- **操作**: 进入首页，观察模块状态卡片
- **预期结果**:
  - 卡片背景为红色（#B71C1C）
  - 显示 Report 图标
  - 标题显示"模块未激活"
  - 摘要显示"请在 LSPosed 中启用模块并勾选背屏中心"

#### TC-HOME-004: Root 权限正常 - 绿色卡片

- **前置条件**: 设备已 Root 且已授权
- **操作**: 进入首页，等待 Root 检测完成
- **预期结果**:
  - Root 状态卡片背景为绿色（#1B5E20）
  - 显示 Ok 图标
  - 标题显示"Root 权限正常"
  - 摘要显示"保存后可自动重启背屏"

#### TC-HOME-005: 无 Root 权限 - 红色卡片

- **前置条件**: 设备未 Root 或未授权
- **操作**: 进入首页，等待 Root 检测完成
- **预期结果**:
  - Root 状态卡片背景为红色（#B71C1C）
  - 显示 Report 图标
  - 标题显示"无 Root 权限"
  - 摘要显示"请授予 Root 权限"

#### TC-HOME-006: Root 检测中 - 灰色卡片

- **前置条件**: 应用刚启动，Root 检测尚未完成（hasRoot 为 null）
- **操作**: 观察首页 Root 状态卡片
- **预期结果**:
  - Root 状态卡片背景为灰色（#424242）
  - 显示 Report 图标
  - 标题显示"正在检测 Root..."
  - 无摘要文本

### 更新通告弹窗

#### TC-HOME-007: 首次安装后弹出更新通告

- **前置条件**: 首次安装或清除数据后（`lastSeenVersion` 为 0）
- **操作**: 启动应用
- **预期结果**:
  - 弹出更新通告弹窗
  - 标题显示当前版本号
  - 显示更新日志内容
  - 底部有 QQ 群入口卡片和"知道了"按钮

#### TC-HOME-008: 点击"知道了"关闭弹窗

- **前置条件**: 更新通告弹窗已弹出
- **操作**: 点击"知道了"按钮
- **预期结果**:
  - 弹窗关闭
  - 重启应用后不再弹出（Release 构建）

#### TC-HOME-009: 点击弹窗外部关闭

- **前置条件**: 更新通告弹窗已弹出
- **操作**: 点击弹窗外部区域
- **预期结果**:
  - 弹窗关闭
  - 行为与点击"知道了"一致

#### TC-HOME-010: QQ 群入口跳转

- **前置条件**: 更新通告弹窗已弹出
- **操作**: 点击"加入 QQ 群反馈交流"卡片
- **预期结果**:
  - 跳转到 QQ 加群链接

#### TC-HOME-010a: 爱发电入口跳转

- **前置条件**: 更新通告弹窗已弹出
- **操作**: 点击"爱发电支持"卡片
- **预期结果**:
  - 跳转到 `https://ifdian.net/a/janus`
  - 卡片右侧显示链接文字 `ifdian.net/a/janus`

#### TC-HOME-011: 版本更新后再次弹出

- **前置条件**: 之前的版本已关闭过弹窗（`lastSeenVersion` = 旧版本号）
- **操作**: 安装新版本后启动应用
- **预期结果**:
  - 弹出新版本的更新通告弹窗
